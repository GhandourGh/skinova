# Quick Start Guide - Standalone Executable

## For Developers (Building)

1. **Build the executable**:
   ```bash
   ./build_executable.sh    # Mac/Linux
   # OR
   build_executable.bat     # Windows
   ```

2. **Find your executable**: `dist/SkinovaClinic` (or `.exe` on Windows)

3. **Test it**: Double-click to run

4. **Distribute**: Send the executable file to your client

## For Clients (Using)

1. **Double-click** `SkinovaClinic.exe` (or `SkinovaClinic` on Mac/Linux)

2. **Wait** 10-30 seconds for startup

3. **Browser opens automatically** - log in and use!

4. **Stop**: Press CTRL+C in the terminal window

## Important Notes

- **No Python needed** - everything is bundled!
- **Data location**: See CLIENT_GUIDE.md for where data is stored
- **First admin**: You may need to create the first admin account before building, or provide instructions
