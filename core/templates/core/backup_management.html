How to Use Backups

1. Click "Create Backup Now" to create a complete backup of your system.
2. Download the backup file and save it to a safe location (USB drive, cloud storage, etc.).
3. To restore on a new system, copy the backup file and run: python manage.py restore_system backup_file.zip
4. Backups include: Database, all data, images, logos, and custom styles.
